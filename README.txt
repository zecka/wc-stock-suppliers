== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wc-stock-suppliers` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPres
3. Install and activate advanced custom field pro
4. Go to Stock Suppliers->settings and fill the form
